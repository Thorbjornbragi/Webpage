Project 1 - This Website

Sorry, this one is in Icelandic...

1. Skref

Fyrsta skrefið í verkefninu var að kynna sér hvernig ætti að setja upp vefsíðu með html. Ég fór vel yfir verkefnalýsinguna þar sem hugbúnaður og kennslumyndbönd voru sett fram til þess að hægt væri að átta sig á hvernig maður fer að þessu. Ég setti upp GitHub aðgang til að byrja með og sótti Bracets til þess að geta hafist handa við html skránna.

Næst var svo að velja hvernig vefsíðu ég vildi gera. Ég fór yfir nokkur templates frá vefsíðunni "https://html5up.net/" þar fann ég vefsíðu Forty (Demo má finna hér: "https://html5up.net/forty") sem mér finnst alveg geggjuð. En þar sem ég var ennþá alveg týndur á því hvernig ég ætti að hefjast handa þá fór ég yfir myndbönd frá kennara til að aðstoða mig, myndskeiðin má finna hér: "https://www.youtube.com/playlist?list=PLGuNTpc293hpIpAVSJ3-SgNolCTa1BfJE".

2. Skref

Nú tók við að skrifa niður html kóðann til þess að setja upp vefsíðuna. Ég hafði ekki unnið með html áður og því þurfti ég að lesa aðeins yfir mismunandi grunna frá vefsíðunni til þess að geta skilið hvernig allt ætti að vera sett upp. Ég byrjaði á að hanna forsíðuna til þess að fá betri hugmynd um hvert vefsíðan ætti að fara. Ég vildi hafa mikið af myndum og hafa vefsíðuna eins myndræna og ég gat án þess að það væri of mikið. Ég vildi hafa áhugamálin mín til að hafa vefsíðuna persónulega til þess að auðvelt væri að kynnast mér. Ég hafði mjög gaman af því að fara yfir gamlar myndir til þess að setja á vefsíðuna. Ég þurfti einnig að uppfæra ferilskránna mína til þess að birta á vefsíðunni.

Ég þurfti að fara mikið fram og til baka til þess að hafa vefsíðuna eins og ég vildi, mikið að breyta og bæta og fá skoðanir frá öðrum. Allt gekk að óskum en tók sinn tíma að setja upp vefsíðuna.

3. Skref

Nú er að birta vefsíðuna. Þá þurfti ég að sækja git í gegnum "https://git-scm.com/". Til þess þurfti ég að sækja homebrew í gegnum terminal þar sem ég er í Mac. Þar fékk ég hjálp frá Youtube frá "https://www.youtube.com/watch?v=hMEyBtsuAJE". Þaðan þurfti ég svo að sækja Git í gegnum Terminal með Homebrew.

Þegar Homebrew og Git var komið upp í tölvunni var næst að remote hlaða inn html skránum inn í Github í gegnum Terminal. Þetta tók sinn tíma og var rosalega mikið vesen. Fyrst þurfi að gera Git init á möppuna og stofna Master file til þess að ýta inn í Github. Til að logga sig inn í Github með API þurfi að gera sér Token til þess að ná að tengjast Remote við Github. En að lokum tókst þetta og vefsíðan komst í loftið. Til þess að bæta þessum texta við þurfi ég að pull-a með git pull í Terminal og push-a breyttum skrám, það gekk vel og auðvelt að bæta við þessum texta og myndum með, í fyrsta skipti sem ég var að uppfæra en annað skiptið var önnur saga.

Alltaf kom upp villa hvað sem ég reyndi, en ég restartaði tölvunni endurtengdi við Github og þá loksins tóks að uppfæra síðuna.
